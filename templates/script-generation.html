Script Generation

Script Generation is a sequence to vector model that generates new dialgoues with previous words as a context. Here this is implemented using LSTMs. The network has been trained with a total of around 50000 sequences with 50 words each. To use this functionality 50 words must be provided as a context for the model to complete and generate new dialogues. Create your dialgoues now!
